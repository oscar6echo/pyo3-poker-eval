# Poker Eval

Python/Rust poker eval package.

```sh
# prerequisite
mm activate work
pip install build

# dev install
pip install -e .

# watch 
cargo watch --watch rust -- pip install -e .

# test
pytest

# install
pip install .

# build
python -m build --sdist --wheel
```
